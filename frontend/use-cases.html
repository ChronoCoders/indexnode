DeFi Analytics

Real-Time Trading Analytics

Index swap events from Uniswap, SushiSwap, and other DEXs to provide traders with verifiable, real-time market data. Every price point and volume metric is cryptographically proven.

  • Track token swaps, liquidity adds/removes, and price impact
  • Real-time WebSocket subscriptions for live trading data
  • Verifiable historical data for backtesting strategies

Query DEX Swaps
query DexSwaps {
  blockchainEvents(
    contractAddress: "0x..."
    eventName: "Swap"
  ) {
    tokenIn
    tokenOut
    amountIn
    amountOut
    timestamp
    merkleProof
  }
}
Track NFT Transfers
subscription NFTTransfers {
  blockchainEventAdded(
    contractAddress: "0x..."
    eventName: "Transfer"

    tokenId

    ipfsUrl
  }
}
