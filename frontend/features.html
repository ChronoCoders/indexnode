Everything you need to build trustless blockchain applications with verifiable data

On-Chain Timestamping

Every indexed event receives an immutable, cryptographically verifiable timestamp stored on Ethereum or Polygon. No trust required.

  • SHA-256 content hashing for all indexed events
  • Smart contract event emission on Ethereum and Polygon
  • Verifiable block number and transaction hash for every timestamp
Timestamp Event
event ContentHashed(
  bytes32 indexed contentHash,
  uint256 blockNumber,
  uint256 timestamp
);

// Emitted on-chain
emit ContentHashed(
  0x3d4f...,
  18524031,
  1706745600
);

IPFS Storage

Decentralized, content-addressed storage with automatic pinning via Pinata. Your data lives permanently on the distributed web.

  • Automatic backup for all indexed events on Pro and Enterprise
  • Content-addressed retrieval via CID
  • 100GB storage on Pro tier, unlimited on Enterprise

GraphQL API

Type-safe, modern API with real-time WebSocket subscriptions. Query exactly the data you need, nothing more, nothing less.

  • WebSocket subscriptions for real-time event streaming
  • Cursor-based pagination for efficient data fetching
  • Type-safe schema with full introspection support
